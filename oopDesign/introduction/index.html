Object-Oriented Programming (OOP) is a programming paradigm. A programming paradigm guides programmers to analyze programming problems, and structure programming solutions, in a specific way.

Programming languages have traditionally divided the world into two parts‚Äîdata and operations on data. Data is static and immutable, except as the operations may change it. In an object-oriented programming language, objects and object interactions are the basic elements of design.
