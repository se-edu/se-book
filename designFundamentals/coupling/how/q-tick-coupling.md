Which of these indicate A is coupled to B?

- [ ] a. component A has access to the internal structure of component B.
- [ ] b. components A and B are written by the same developer.
- [ ] c. component A calls component B.
- [ ] d. component A receives an object of component B as a parameter.
- [ ] e. component A inherits from component B.
- [ ] f. component B inherits from component A.
- [ ] g. components A and B have to follow the same data format or communication protocol.

<panel type="seamless" header="{{ icon_A }} Answer" minimized>

(a)~~(b)~~(c)(d)(e)~~(f)~~(g)

Explanation: Being written by the same developer does not imply coupling. The parent class is not coupled to the child class, but the child class is coupled to the parent class.
