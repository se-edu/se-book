You can use the `stage` or the `add` command (they are synonyms, `add` is the more popular choice) to stage files.

```{.no-line-numbers}
$ git add fruits.txt
$ git status
```
{{ icon_output }}
```{.no-line-numbers}
# On branch master
#
# No commits yet
#
# Changes to be committed:
#   (use "git rm --cached <file>..." to unstage)
#
#       new file:   fruits.txt
#
```
